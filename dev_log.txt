**********************
*** 09.01.2016 *******
**********************
Регистрация будет осуществляться так:

HomeController метод Index -
метод с формой для регистрации на сайте в качестве пользователя.

После POST-отправки данных вызывается метод Index, включающий объект 
модели User в качестве пользователя. Этот метод перенаправляет на 
результат действия метода HomeController.Register(), который должен:

а) внести данные в БД если они валидны.
б) вернуть на представление с методом Index.
